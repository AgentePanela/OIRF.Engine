Project ORIF

### OIRF Engine: (Temporary Name)
Is a C#-made 2D game engine made using the Monogame framework, inspired by the Robust Toolbox from SS14. This engine uses the ECS architecture and the prototypes architecture from Robust Toolbox.

### Requeriments:
- Visual C++ Redistributable Packages for Visual Studio 2013 (MonoGame Content Pipeline shader compiller need the 2013 installed to compile shaders)
